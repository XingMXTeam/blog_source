## 概念

教堂： 商业软件
集市： 开源

+ 学术型协议： 修改需要注明原著和免责声明
  + BSD Apache MIT PHP
+ CopyLeft/病毒型协议： 学术型要求+衍生作品必须也开源
  + GPL LGPL AGPL
  
衍生作品：修改过代码或者一起编译或者GPL静态链接
非衍生作品： 不同层面运行；开源和我的程序独立运行没有交互；远程调用

+ 发行：修改了源代码但是没有发行就不必公开修改过的源码
  + 下载到用户端；
